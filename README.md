# Ikshana – Skin Remedy App

An Android application designed to provide personalized skin care remedies based on user inputs, focusing on usability and performance.

## Features
- Personalized skin remedy suggestions
- Clean and responsive Android UI
- Smooth navigation and optimized layouts

## Tech Stack
- Java
- XML
- Android Studio
- Backend APIs

## Overview
The application collects user inputs related to skin concerns and delivers tailored remedy recommendations with an emphasis on user experience and app stability.
